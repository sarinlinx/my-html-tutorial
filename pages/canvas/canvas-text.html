Drawing Text

Top methods to draw text.

font - defines the font properties for the text

fillText(text,x,y) - draws "filled" text on the canvas. X and Y position the text.

strokeText(text,x,y) - draws text on the canvas (no fill)

<canvas id="myCanvas4" width="200" height="100" style="border:1px solid #000000;">

var canvas = document.getElementById("myCanvas");
var ctx = canvas.getContext("2d");
ctx.font = "30px Arial";
ctx.fillText("Hello World",10,50);

Add color and center the text

Add fillStyle = "red" to fill in the text.

To fillText, center with canvas.width/2, canvas.height/2.

<canvas id="myCanvas6" width="200" height="100" style="border:1px solid #000000;">

var canvas = document.getElementById("myCanvas6");
var ctx = canvas.getContext("2d");
ctx.font = "30px Comic Sans MS";
ctx.fillStyle = "red";
ctx.textAlign = "center";
ctx.fillText("Hello World", canvas.width/2, canvas.height/2);
