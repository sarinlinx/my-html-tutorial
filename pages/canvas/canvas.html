Canvas overview

Canvas is an HTML element: <canvas>. It is only a container for graphics.

JavaScript is used to draw elements within the Canvas element.

Canvas can be used for:

  • drawing colorful text with animation
  • graphs
  • interactivly respond to JavaScript events

Simple example

Basic HTMl example

The Canvas element must have an id attribute so it can be referred to by JavaScript.

Width and Height are necessary to define the size of the Canvas.

<canvas id="myCanvas" width="200" height="100"></canvas>

Adding a border

There's no border by default. Use a style attribute to add one.

<canvas id="myCanvas" width="200" height="100" style="border:1px solid #000000;"></canvas>

Drawing

Create the HTML Canvas element

First, create the HTML element.

<canvas id="myCanvas" width="200" height="100"></canvas>

Locate the Canvas element in JavaScript

JavaScript is used to draw. The first step is to locate it.

var canvas = document.getElementById("myCanvas");

Create a drawing Object

The getContext() is a built-in HTML object, with properties and methods for drawing.

var ctx = canvas.getContext("2d");

Draw

Part 1

Set the fill style of the drawing object to a color.

NOTE: This must come BEFORE the drawing of the Object.

The fillStyle property can be a CSS color, a gradient, or a pattern. The default fillStyle is black.

This fills the rectangle you just drew with the color red.

ctx.fillStyle = "#FF0000";

Part 2

Use use fillRect() to draw a rectangle.

ctx.fillRect(0,0,150,75);

Example

Coordinates

The upper-left corner of the canvas has the coordinates (0,0).

Drawing a rectangle would use fillRect(0,0,150,75).

This means: Start at the upper-left corner (0,0) and draw a 150x75 pixels rectangle.

Draw a line

First locate the element and create a drawing Object.

var canvas = document.getElementById("myCanvas");
var ctx = canvas.getContext("2d");

Next, use moveTo(x,y) to define the starting point.

Then use lineTo(x,y) to define the ending point.

Finally, to actually draw the line, use a method such as stroke().

var canvas = document.getElementById("myCanvas");
var ctx = canvas.getContext("2d");
ctx.moveTo(0,0);
ctx.lineTo(200,100);
ctx.stroke();

Example

Draw a Circle

Use beginPath() to begin the path.

arc(x,y,r,startangle,endangle) - creates an arc/curve.

The x and y parameters define the x- and y-coordinates of the center of the circle. The r parameter defines the radius of the circle.

To create a circle with arc(): Set start angle to 0 and end angle to 2*Math.PI.

<canvas id="myCanvas3" width="200" height="100" style="border:1px solid #000000;">

var canvas = document.getElementById("myCanvas3");
var ctx = canvas.getContext("2d");
ctx.beginPath();
ctx.arc(95,50,40,0,2*Math.PI);
ctx.stroke();
